Getting Started

When you log in to MicroMain Mobile you are presented with an initial page that shows existing work orders. The Assignment column (Indicated by an A in the title) is typically displayed in the left column. In some cases the assignment column may not be displayed. An asterisk in the assignment column indicates the work order is assigned to the login user/labor.  If the left column is blank, the work order is unassigned, or assigned to someone else. The information displayed in the assignment column is controlled by the MicroMain Control Center program.

 

Up to ten work orders are displayed on each page. If there are more than 10 work orders, use the navigation links at the bottom of the work order page (First, Last, Next, Previous) to move between pages. Click on the title of a column to toggle the work order list between ascending or descending order based on the data in the column you select.

Navigating and Saving Data

Use the breadcrumbs on the Mobile Web interface to move between pages for best results. Avoid using your browser’s Back and Forward buttons to prevent possible loss of data on the current page. In most cases, your data is saved when you navigate away from a page or a region using the breadcrumbs in the Mobile Web application. However, there are some exceptions to this rule. For the following types of data, you must perform the specific actions indicated to save your data:

·           CREATING A WORK ORDER: You must click Create WO to save the data. Clicking Home or Back will abort the data you have entered for the new work order.

·           ADDING A DESCRIPTION TO A WORK ORDER: You must click Add to Description to add a description to the work order. The descriptions you add cannot be deleted.

·           ADDING A COMMENT TO A WORK ORDER: You must click Add to Comments to add comments to the work order. The comments you add cannot be deleted.

·           UPDATING METERS: You must click Update to update the reading.

Creating a Work Order

1. Click New WO from the home page or initial login page.

2. Select a service from the drop-down list, or select Other to add a new service. NOTE: If you select Other, you can return to the list view by clicking the List link at the top of the work order page.

3. Select a property from the drop-down list.

4. Select a building from the drop-down list. NOTE: The buildings displayed are dependent upon the property selected in the previous step.

5. Select an asset group from the drop-down menu.

6. Select an asset from the drop-down menu. NOTE: The assets displayed are dependent upon the property, building, and asset group selected in the previous steps.

7. Click Create WO to save the work order.

Return to Contents

 

Updating a Work Order

1. Click on the desired work order to open it.

2. Click the work order tab to expand it.

3. Choose a service from the drop down list, or select Other to enter a new service in the text window.

4. The Property and Asset fields are populated when the work order is created and cannot be updated.

5. Click Change Status to open the status selection menu.

6. Set the Status field to Open, Completed, or Hold using the drop-down menu. NOTE: The choices in the Status field are dependent on the current status of the work order.

7. Click OK.

8. Open the Details tab and perform the following actions:

a. Select a sub-status from the drop-down list.

b. Select a shop from the drop-down list.

c. Enter a location. The data you enter will be saved when you navigate away from the page.

d. Select a priority from the drop-down list, with 0 being the highest, and 9 being the lowest.

e. Click on the Due field and select the due date from the calendar.

f. The Type, Requester, and Requester’s Phone are view-only fields and cannot be updated in MicroMain Mobile. Changes to the fields must be made in MicroMain Maintenance.

g. Select the primary and secondary failures from the drop-down lists.

h. Enter the downtime in hours and tenths or hundredths of hours. For example, 5.25 hours indicates five hours and 15 minutes.

i. Select an account from the drop-down menu.

j. The type of work is displayed in the checkboxes at the bottom of the Details tab. These checkboxes are read-only fields that are controlled through the MicroMain Maintenance program, and cannot be updated by mobile users.

9. Click Home or Back to save the changes.

Adding Labor Manually

1. From the work order page, click on the Labor link.

2. Click on the Add link at the top of the page.

3. Select a labor asset from the drop-down list.

4. Click on the Date field and select a date from the pop-up calendar.

5. Click on the Time In button to enter the current time as start time (or you can manually enter the start time).

6. Click on the Time Out button to enter the current time as the end time (or you can manually enter the end time).
NOTE: If the time out is expected to be past midnight, add new labor after midnight to log the additional time.

7. Click on the Calculate button to display the amount of time spent.

8. Click Home or Back to save the data, or click Cancel to abort the data.

Return to Contents

Adding Labor by Scanning

NOTE: Your mobile device must be configured and paired with a Bluetooth barcode scanner before performing this procedure.

1. From the work order page, click on the Labor link.

2. Click on the Scan link at the top of the page.

3. Ensure that the cursor is anchored in the Scan Barcode (Control Number) field.

4. Activate your Bluetooth barcode scanner to load the barcode into the field. NOTE: The scanner assumes keyboard control from the mobile device when activated. You must power down the scanner to return keyboard control to your mobile device after scanning the barcode.

5. Click on the Date field and select a date from the pop-up calendar.

6. Click on the Time In button to enter the current time as start time (or you can manually enter the start time).

7. Click on the Time Out button to enter the current time as the end time (or you can manually enter the end time).
NOTE: If the time out is expected to be past midnight, add new labor after midnight to log the additional time.

8. Click on the Calculate button to display the amount of time spent.

9. Click Home or Back  to save the data, or click Cancel to abort the data.

Return to Contents

 

Updating Labor

1. From the work order page, click on the Labor link.

2. Click on the row of the labor item you wish to update.

3. Modify the Date, Time In, and Time Out fields as needed.

4. Click the Calculate button to update the regular hours.

5. Click Home or Back  to save the changes.

Return to Contents

 

Adding Parts Manually

1. From the work order page, click on the Parts link.

2. Click on the Add link at the top of the page.

3. Select a location from the drop-down menu.

5. Enter the actual quantity of the part to add to the work order.

6. Click Home or Back  to save the data, or click Cancel to abort the data.

Viewing and Updating Inspection Points

1. From the work order page, click on the Points link. NOTE: If inspection points were not assigned to the work order in MicroMain Maintenance, the Inspection Points List will be empty.

1.       2. The Inspection Points List shows all the inspection points assigned to the work order. The Point column lists the inspection point. Group lists the inspection point group. Status lists the current status of the inspection point (Pass, Fail, or Unvisited). Sort lists the sort priority.

2.       3. Sort the list by clicking on the heading of the column you wish to sort by.

3.       4. Click an inspection point in the Inspection Points List to manually access the Edit Point form.

4.       5. Use the controls on the Edit Point form to view or modify: Status, Visited Date, Visited Time, Rating, Measurement, Action, and Comments. NOTE: Modifying the Status will cause Visited Date and Visited Time to automatically populate with the current values. Visited Date and Visited Time can also be manually edited.

Return to Contents

 

Viewing and Updating Inspection Points by Scanning

1. From the work order page, click on the Points link. NOTE: If inspection points were not assigned to the work order in MicroMain Maintenance, the Inspection Points List will be empty.

2. Ensure that the cursor is anchored in the Scan Barcode (Control Number) field.

3. Activate your Bluetooth barcode scanner to load the barcode into the field. NOTE: The scanner assumes keyboard control from the mobile device when activated. You must power down the scanner to return keyboard control to your mobile device after scanning the barcode.

4. If the status of an inspection point is Unvisited, scanning will change the status to Passed and the Visited Date and Visited Time will automatically populate with the current values. NOTE: If the status is already set to Passed or Failed, scanning will cause the Edit Point form to open. You may then use the controls on the Edit Point form to view or modify: Status, Visited Date, Visited Time, Rating, Measurement, Action, and Comments.

Return to Contents

6.       Viewing and Updating Meters
